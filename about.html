Rename File in Ubuntu Using the mv Command Before using the ‘mv’ command, you should know how it works on your system. The basic syntax of the ‘mv’ command is given below: mv [options] [file-source] [Destination] The most popular ‘mv’ command options are provided below: -f – Displays no message or alerts before overwriting a file name. -i – Displays prompt confirmation or warning messages before renaming a file. -u – It moves a file if the file does not exist on the specified destination or in case of a new file. /p>
